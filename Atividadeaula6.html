Sobre HTML

HTML significa "Hypertext Markup Language" (Linguagem de Marcação de Hipertexto, em português) e é a linguagem padrão para criação de páginas web. O HTML é usado para estruturar o conteúdo de uma página web, permitindo que os navegadores da internet exibam texto, imagens, vídeos e outros elementos de forma organizada e formatada. Ele é composto por um conjunto de etiquetas (também chamadas de tags ou marcadores), que indicam como o conteúdo deve ser apresentado.

Sobre HTML Semântico

HTML semântico refere-se à prática de usar elementos HTML de maneira significativa e descritiva, com o objetivo de fornecer uma estrutura e um significado claros para o conteúdo de uma página web. Em vez de usar elementos HTML apenas para formatar o conteúdo visualmente, o HTML semântico enfatiza o uso apropriado de elementos que transmitam o propósito e a estrutura do conteúdo. Isso ajuda os mecanismos de busca a entenderem melhor o conteúdo da página e também melhora a acessibilidade para pessoas que usam leitores de tela e outros dispositivos assistivos.
